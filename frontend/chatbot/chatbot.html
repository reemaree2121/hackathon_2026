Apex Campus AI Assistant

This standalone page demonstrates the embedded AI Chatbot widget. Notice the floating AI button at the bottom right corner of the browser window.

How to integrate into any HTML/CSS/JS page:

Copy the files into your project folder `/chatbot/`, then include these script tags before the closing </body> tag:

<!-- 1. Include CSS in head --> <link rel="stylesheet" href="/chatbot/chatbot.css" /> <!-- 2. Include JS ES Modules at bottom of body --> <script type="module"> import { CampusChatbot } from '/chatbot/chatbot.js'; // Initialize Chatbot with configuration CampusChatbot.init({ provider: 'gemini', // 'gemini' | 'openai' | 'openrouter' | 'local' | 'fallback' apiKey: '', // Added here or handled via server proxy activePage: 'Dashboard', theme: 'light' }); </script>
